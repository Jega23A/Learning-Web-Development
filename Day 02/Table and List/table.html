Table Header

'th' stands for table header.Sometimes we want our cells to be table header cells.In those cases use the 'th' tag instead of the 'td' tag.

Table Rows

'tr' stands for table row.Each table row starts with a 'tr' tag and ends with a '/tr' tag

Table Data

'td' stands for table data.Each table cell is defined by a 'td' tag.
