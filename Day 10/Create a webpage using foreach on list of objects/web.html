Book List

Here we create a webpage using foreach on a list of objects, I'll provide a simple example using HTML, CSS, and JavaScript.
Let's say we have a list of objects representing books:
const books = [
{ title: "Book 1", author: "Author 1" },
{ title: "Book 2", author: "Author 2" },
{ title: "Book 3", author: "Author 3" },
];
Here we display this list on a webpage using 'foreach'.

In this example, we:
1. Create an HTML structure with a container (#book-list) for the book list.
2. Define CSS styles for the book list and individual books.
3. Use JavaScript to iterate over the books array using foreach.
4. For each book, create a new div element, add the "book" class, and set its inner HTML to display the title and author.
5. Append each book element to the #book-list container.
