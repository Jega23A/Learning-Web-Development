The box-shadow Property

This is a div element with a box-shadow

In its simplest use, you only specify a horizontal and a vertical shadow.
The default color of the shadow is the current text color.
Here we specify the color parameter as lightblue which defines the color of the shadow.


Other properties we can include are 'Blur Effect', 'Spread Radius', 'Inset Parameter' and 'Multiple Shadows'.
