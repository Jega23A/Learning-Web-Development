The Inline Element

An Inline element does not start on a new line

Some of the Inline Elements are: span,a,b,br

The SPAN Element

This is an inline span Hello World element inside a paragraph.

The SPAN element is an inline element, and will not start on a new line and only takes up as much width as necessary.

The a Element

Visit youtube.com!

The 'a' tag defines a hyperlink, which is used to link from one page to another

The most important attribute of the 'a' element is the href attribute, which indicates the link's destination

By default,links will appear as follows in all browswer's:

*An unvisited link is underlined as blue

*A visited link is underlined as purple

*An active link is underlined as red
